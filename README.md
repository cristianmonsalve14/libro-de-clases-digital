# Plataforma de Libro de Clases Digital

Este proyecto es una plataforma de gestión académica desarrollada como parte del ramo Fullstack III. La arquitectura está basada en microservicios con Spring Boot para el backend y un frontend desacoplado.

## Estado del Proyecto

Actualmente, el proyecto se encuentra en la fase de diseño e implementación inicial del backend.

- **Base de Datos**: El esquema inicial en PostgreSQL ha sido definido y refinado.
- **Backend**: Se ha iniciado la creación del servicio de autenticación (`auth-service`).
- **Frontend**: La estructura de carpetas está creada, pendiente de desarrollo.

## Próximos Pasos

1.  Completar el scaffolding del `auth-service`.
2.  Crear las entidades JPA y los repositorios.
3.  Implementar la lógica de registro y login de usuarios.
